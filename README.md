# Blinkit-Sales-Data-Analysis-Excel-Project-
This project analyzes sales data using Microsoft Excel to uncover key business insights such as revenue trends, customer behavior, and product performance. The analysis is performed using Excel features like Pivot Tables, Charts, and Data Cleaning techniques.

Project Overview:

The goal of this project is to transform raw sales data into meaningful insights using Excel. It demonstrates practical data analysis skills that are widely used in business environments.

Key objectives:

* Analyze sales performance
* Identify top-performing products
* Understand customer trends
* Track revenue patterns

Dataset Information:

The project uses the following datasets:

* **customers.xlsx** → Customer details
* **products.xlsx** → Product information
* **orders.xlsx** → Sales transactions

Features:

* Data cleaning and preprocessing
* Pivot Tables for data summarization
* Interactive dashboards
* Charts and visualizations
* KPI calculations (Revenue, Orders, etc.)
* Trend analysis

Analysis Performed:

* Total revenue calculation
* Monthly sales trends
* Top-selling products
* Category-wise performance
* Customer segmentation
* City-wise sales analysis

Dashboard Highlights:

* Revenue overview
* Sales trends over time
* Top products and categories
* Customer distribution
* Order patterns

Tools Used:

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Data Cleaning Functions

How to Use:

1. Download the Excel files
2. Open **Blinkit Sales Data Analysis.xlsx**
3. Navigate through different sheets:

   * Raw Data
   * Pivot Tables
   * Dashboard
4. Use filters/slicers to interact with the dashboard
